CODE SAMPLES

Find the digital root - the recursive sum of all the digits in a number:

function digital_root(n) {
    let result = 0;
    while (n) {
        result += n % 10;
        n = Math.floor(n / 10);
    }
    return result < 10 ? result : digital_root(result);
}
digital_root(16) // result = 7
digital_root(456) // result = 6

Array difference - subtract one list from another and return the result:

function arrayDiff(a, b) {
    return a.filter(el => !b.includes(el));
}
arrayDiff([1,2],[1]); // result = [2]
arrayDiff([1,2,2,2,3],[2]); // result = [1,3]
